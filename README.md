# 军团冲击 (Legion Rush)

基于《放置军团》(Gubulin) 的完整复刻 + 玩法优化项目。

## 项目概述

复刻原版放置军团的核心玩法，并在其基础上融入 2024-2025 年最流行的手游机制进行优化升级。

## 技术栈

- **游戏引擎**：Cocos Creator 3.8.8（2D 模板）
- **开发语言**：TypeScript
- **目标平台**：微信小游戏 / iOS / Android（一套代码多端发布）

## 核心玩法

- 100v100 自动战斗，约30秒一局
- 战前排兵布阵，战斗自动进行
- 四大种族（哥布林 / 魔族 / 人族 / 亡灵）+ 中立兵种
- 兵种养成：等级、品质进阶、装备宝物
- 推图、副本、竞技场、抽卡等完整系统

## 优化亮点（新增）

1. **种族羁绊系统** - 类似金铲铲的羁绊组合玩法
2. **手动必杀技** - A+品质解锁，半自动战斗增加操作感
3. **肉鸽推图模式** - Slay the Spire式随机路径 + 三选一Buff
4. **兵种进化分支** - 紫色品质二选一进化方向
5. **赛季制 + 通行证** - 6周一赛季，持续内容更新
6. **公会系统 + 领土战** - 社交协作 + 领土争夺
7. **融合系统** - 3合1升级 + 跨种融合新兵种
8. **无尽爬塔** - 无限层数 + 全服排行榜
9. **剧情系统 + 好感度** - 角色故事和情感连接
10. **实时PVP** - 天梯匹配 + 段位制 + 观战
