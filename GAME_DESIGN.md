# 军团冲击 (Legion Rush) - 游戏设计文档

> 基于《放置军团》(Gubulin) 核心玩法的复刻 + 优化升级项目

## 一、游戏基本信息

- **游戏名称**：军团冲击（Legion Rush）
- **原作参考**：放置军团（Gubulin）- 上海信行软件有限公司
- **游戏类型**：放置类模拟经营 + 即时战略（RTS）+ 策略（SLG）手游
- **核心卖点**：100v100，胜负30秒，"恶人"主角
- **游戏引擎**：Cocos Creator 3.8.8（2D）
- **开发语言**：TypeScript
- **目标平台**：微信小游戏 / iOS / Android
- **原作官网**：https://m.gubulin.com/

## 二、核心玩法系统

### 2.1 战斗系统
- **规模**：100v100 单位自动战斗，战斗时间约30秒
- **操控方式**：玩家在战前部署军团阵型/站位，战斗自动进行
- **核心循环**：推图 -> 获取资源 -> 升级军团 -> 推进更高关卡
- **战斗要素**：
  - 兵种有等级、品质（绿->蓝->紫->金->金1->金2->金3）
  - 兵种有类型区分：坦克、输出、控制、刺客、辅助
  - 技能自动释放，有CD机制
  - 人数越多，某些种族（如哥布林）的加成越高
- **离线收益**：上限8小时（月卡后10小时）

### 2.2 军团（兵种）系统

#### 2.2.1 四大种族 + 中立

**哥布林族（推荐开局种族）**

#### 2.2.2 兵种品质与进阶
- **品质路线**：绿 -> 蓝 -> 紫 -> 金 -> 金1 -> 金2 -> 金3（抽卡可直接到紫）
- **升品材料**：需要对应的魂石碎片 + 进阶卷轴
- **品质等级影响**：A+ 质变节点（很多兵种A+解锁强力被动）
- **双孔**：部分强力兵种（如炮手）需要"双孔"才能发挥真正实力

#### 2.2.3 品质评级
- 从低到高：B -> B+ -> A -> A+ -> S -> S+ 等
- 评级由基础属性+装备+宝物综合决定

### 2.3 阵型/布阵系统
- 玩家在战前排布军团的位置（站位影响战斗走向）
- 不同关卡可能需要不同阵型
- 可以查看录像学习其他玩家的站位
- 可通过圣器解锁更多阵型

### 2.4 推荐阵容（当前版本最优解）

**推图主力阵容**（哥布林核心）：
1. 影魔 - 全图控制+输出
2. 哥布林炮手 - 主力AOE输出
3. 哥布林骑士 - 前排+控制+切后排
4. 哥布林预言家 - 减益+吸引火力
5. 哥布林勇士 - 分身+触发人数buff

**培养优先级**：炮 > 影魔 > 猪骑 > 哥布林 > 预言家 > 哥布林勇士

## 三、PVE关卡系统

### 3.1 主线推图
- 章节制（第一章、第二章...），每章30关
- **三选一奖励**：通关每关从3个随机奖励选1个（可退出重刷）
  - 普通关（1,2,3,4关等）：可掉绿色魂石
  - 小Boss关（5,10,15,25关）：可掉蓝色魂石
  - 大Boss关（20,30关）：可掉紫色魂石
- **关卡奖励优先级**：进阶证 > 瓶盖 > 沙漏（普通关）；主力魂石 > 筹码 > 稀有魂石（Boss关）

### 3.2 五大副本

#### 副本1：限时征讨（通关2-10解锁）
- **产出**：经验书、氪晶、沙漏、魂石、军团、中级进阶书等（游戏产出大头）
- **机制**：
  - 每双数整点获得1次挑战次数，一周最多88次
  - 共12关，前1-6关任意兵种可打
  - 7-12关需通关进阶挑战，只能用指定种族
  - 种族轮换：哥布林 -> 魔族 -> 人族 -> 亡灵（每周轮换）
- **进阶意义**：打进阶7比普通6每周多拿1000+氪晶，进阶8再多1500氪晶

#### 副本2：据点防守（通关2-20解锁）
- **产出**：进阶书/进阶素材
- **机制**：使用远程军团防守3波进攻，输了不扣次数
- **推荐阵容**：炮手、影魔、蛛魔、预言家、骨弓

#### 副本3：连环突击（通关3-10解锁）
- **产出**：魂石、宝箱、沙漏
- **机制**：每天3次，可用60氪晶刷新（推荐刷到5星再打）
- **注意**：补满兵再打，中途可换人换阵型

#### 副本4：宝物副本（通关5-20解锁）
- **产出**：宝物碎片、未鉴定宝物
- **机制**：
  - 每天5个小关获取碎片，3000碎片可10连抽宝物
  - 打Boss提升信号强度，60-70出5星蒙尘宝物
  - 共8层，越高碎片越多
  - **难度根据进入时最高5个战力决定**（先进副本再升级）

#### 副本5：中立副本
- **产出**：中立魂石
- **机制**：每周六日各3次，失败不扣次数
- **目标**：打到中立5以上出紫色魂石

### 3.3 军团征讨
- 四族循环征讨：哥布林打魔族 -> 魔族打人族 -> 人族打亡灵 -> 亡灵打哥布林
- 每周轮换一次
- 需要培养所有种族才能拿满奖励

## 四、养成系统

### 4.1 抽卡系统
- **普通卡池**：前期推图用，直接抽普通池
- **UP卡池**：屯120筹码保底抽（保底机制：两次十连保底一个金色种族）
- **限时活动卡池**：可抽麒麟（中立刺客），约每月出1-2次
- **首抽十连**：开局送十连（必紫），建议刷到猪骑/炮手开局
- **30连抽**：开局30连三选一，建议刷猪骑/炮兵

### 4.2 装备/宝物系统
- **宝物类型**：套装、专属、通用宝物
- **关键宝物**：
  - 哥布林套装：集齐后随机将一个哥布林变Boss
  - 炮手专属：大幅增强炮手输出
  - 亡灵套装：4件质变
  - 蓝胡子玩偶、绿胡子玩偶、神银弩箭：神器级别
- **宝物品质**：1-5星，5星蒙尘宝物需要鉴定
- **宝物主属性**：随机且不可洗
- **鉴定**：需要材料，在卷轴作坊制作

### 4.3 圣器系统（魔王天赋树）
- **三页天赋**：
  - 第一页：优先点满（50% Boss伤害是推图必须）-> 先点魔王经验获取 -> 再点升阶消耗减少
  - 第二页：优先卷轴作坊消耗降低 -> 据点收益增加
  - 第三页：先点两个工人 -> 粉尘采集加倍 -> 概率不消耗原料
- **魔王点**：通过氪晶商店购买、任务获取

### 4.4 建筑系统
- **仓库**：最优先升级
- **兵营**：第二优先，等级越高带兵上限越高
- **地风水之门**：第三优先
- **卷轴加工**：后续升级
- **兽栏**：最后升级

### 4.5 工人系统
- 工人用于采集资源、制作材料
- 前期用4个魔王点优先点出两个古卜林（工人）
- 猪猪存钱罐：存储资源的特殊机制

## 五、经济系统

### 5.1 货币类型
| 货币 | 获取方式 | 主要用途 |
|------|----------|----------|
| 氪晶 | 推图、征讨、任务、充值 | 商店购买、刷新副本、开阵型 |
| 魔晶 | 日常产出 | 升级材料消耗 |
| 筹码 | 推图掉落、商店 | 抽卡 |
| 瓶盖 | 推图掉落 | 升级材料 |
| 竞技场币 | 竞技场排名奖励 | 兑换兽族、筹码等 |
| 宝物碎片 | 宝物副本 | 抽宝物 |

### 5.2 氪晶使用优先级
1. 氪晶商店：筹码、888古卜林、2880魔王点（必买）
2. 连环突击刷新5星
3. 开阵型
4. 不推荐直接抽卡（黑市买筹码打折）

### 5.3 沙漏使用
- **魔晶沙漏**：有需要就用
- **时间沙漏**：短的可以适当用，长的留到后期人口多时用

## 九、技术方案（已确认）

- **目标平台**：微信小游戏 + iOS + Android（一套代码多端发布）
- **游戏引擎**：Cocos Creator 3.8.8（2D TypeScript）
- **项目路径**：`LegionRush/`
- **复刻范围**：完整复刻全部系统
- **优化方向**：新增功能

### 9.1 为什么选 Cocos Creator
- 一套代码，多端发布：微信小游戏 / iOS (Xcode→IPA) / Android (APK) / Web
- 2D 游戏生态成熟，UI系统强大
- TypeScript 开发，类型安全，维护性好
- 国内社区活跃，文档齐全
- 分包加载支持，微信小游戏无压力

### 9.2 各平台注意事项
- **微信小游戏**：主包 < 4MB，总包体 < 20MB（需分包策略），纯 Canvas/WebGL 渲染
- **iOS**：构建 Xcode 工程，需 Apple 开发者账号，需版号
- **Android**：构建 APK/AAB，可上架各应用商店
- **后端**：微信云开发（小游戏）或自建服务器（全平台通用）
- **数据同步**：多端需统一账号体系（微信登录 / 手机号登录）

### 10.2 核心架构模式
- **ECS精简版**：Entity（实体=兵种单位）+ Component（组件=技能、AI、属性）+ System（系统=战斗、渲染）
- **事件驱动**：全局EventBus解耦各系统
- **数据驱动**：所有数值配置外置JSON，热更新友好
- **状态机**：每个单位用FSM管理状态（Idle/Move/Attack/Skill/Dead）
- **对象池**：100v100单位频繁创建销毁，必须用对象池复用

### 10.3 数据存储方案
- **本地存储**：微信小游戏 wx.setStorageSync / wx.getStorageSync
- **云存储**：微信云开发 CloudBase（用户数据同步、防作弊）
- **离线收益计算**：记录上次在线时间戳，上线后差值 × 每秒收益

## 十一、分阶段实施计划

### 第一阶段：基础框架 + 战斗核心（预计2-3周）
**目标**：跑通100v100自动战斗

1. **项目初始化**
   - ~~创建 Cocos Creator 3.x 项目~~（已完成：Cocos Creator 3.8.8，2D模板，项目名 LegionRush）
   - 搭建目录结构和核心框架（EventBus、对象池、配置管理）
   - 集成 TypeScript 严格模式

2. **战斗系统核心**
   - `BattleManager`：管理战斗流程（初始化 → 开战 → 每帧更新 → 结算）
   - `Unit` 类：基础属性（HP/ATK/DEF/SPD/Range）、状态机

### 第七阶段：玩法优化 + 新增系统（预计3-4周）
**目标**：融合2024-2025年最流行的手游机制，全面提升游戏深度和留存

> **设计理念**：原版《放置军团》核心循环优秀但策略深度不足（种族搭配固定、战斗纯自动、内容消耗快）。
> 以下优化围绕三个方向展开：**增加策略深度**、**提升战斗参与感**、**延长内容寿命**。

#### 优化1：种族羁绊系统（核心优化，类似云顶之弈/金铲铲）
**为什么做**：原版仅有"人多势众"一个隐性种族加成，阵容搭配自由度低。羁绊系统是当前最火的策略玩法（金铲铲之战DAU千万级），能极大丰富阵容构建的乐趣。

**设计详情**：
- 每个种族定义 2-3 个羁绊效果，按上场同族兵种数量分 2/4/6 档激活

| 羁绊 | 2体效果 | 4体效果 | 6体效果 |
|------|--------|--------|--------|
| **哥布林·蜂拥** | 全体攻速+10% | 全体攻速+20%，哥布林数量额外+15%攻击 | 召唤额外哥布林小兵（每10秒1波） |
| **魔族·恐惧** | 敌方全体攻击-8% | 敌方全体攻击-15%，被击时5%概率恐惧 | 魔族击杀单位后恢复15%HP |
| **人族·圣盾** | 人族单位受治疗+20% | 人族单位死亡时给友军套护盾 | 每20秒全体人族无敌2秒 |
| **亡灵·不朽** | 亡灵单位复活1次（30%HP） | 亡灵复活后伤害+50% | 亡灵单位每次死亡使全体亡灵永久+2%攻击（本场战斗） |
| **混编·联军** | 上场3个不同种族 | 上场4个不同种族，全属性+12% | 上场5个不同种族，开场释放全体AOE |

- 混编羁绊鼓励玩家突破"纯种阵容"的局限
- 羁绊效果在布阵界面实时预览，方便决策
- **数据结构**：新增 `synergies.json` 配置表

#### 优化2：肉鸽推图模式（Roguelike Campaign）
**为什么做**：肉鸽是2024-2025年增长最快的游戏机制。原版推图是纯线性关卡，容易产生"卡关就没事做"的挫败感。肉鸽模式提供无限可玩性和"再来一把"的驱动力。

**设计详情**：
- 主线推图之外新增 **"异界远征"** 模式（独立入口）
- 采用 Slay the Spire 式地图：每层3-5个节点随机生成路径
- 节点类型：**战斗节点 / 精英战斗 / 商店 / 事件 / Boss / 休息点**
- 每次战斗胜利后 **三选一获得临时Buff**（仅本次远征有效）：
  - 例："全员暴击率+20%"、"哥布林骑士冲锋距离+50%"、"炮手AOE范围+30%"
  - 稀有Buff："复活一名阵亡单位"、"Boss变为友军"
- **死亡惩罚**：远征中全军覆没则结束，根据通关层数结算奖励（魂石碎片、氪晶、专属宝物）
- **每日/每周挑战**：固定种子排行榜，同一随机序列比拼谁打得远
- 新增配置：`roguelike_buffs.json`（200+种随机Buff）、`roguelike_events.json`（随机事件）

#### 优化3：手动必杀技（Semi-Auto Combat）
**为什么做**：纯自动战斗让玩家在30秒内无所事事，降低参与感。允许手动释放必杀技既保留了"放置"的低门槛特性，又给操作型玩家发挥空间。

**设计详情**：
- 每个兵种在品质达到 **A+** 后解锁 **必杀技**（Ultimate）
- 必杀技有独立能量条，通过战斗自动充能（约15-20秒充满）
- 能量满后玩家可 **点击兵种头像手动释放**，或设置为自动释放
- 必杀技示例：
  - 哥布林骑士 → "猪突猛进"：直线冲锋，沿途敌人击飞+眩晕3秒
  - 哥布林炮手 → "饱和轰炸"：全屏AOE，造成300%攻击力的范围伤害
  - 影魔 → "暗影收割"：对全图最低HP单位造成致命伤害（低于30%直接斩杀）
  - 剑魔 → "魔剑领域"：自身周围大范围持续伤害+减伤50%
- 可设置"自动释放策略"：血量低于X%时自动放 / 能量满立即放 / 手动控制
- **PVP影响**：竞技场中手动操作是区分高低手的关键

#### 优化4：兵种进化分支（Branching Evolution）
**为什么做**：原版进阶是纯线性的（绿→蓝→紫→金），缺乏选择。引入分支进化让玩家在同一种兵种上有不同的培养方向，增加配队多样性。

**设计详情**：
- 兵种达到 **紫色品质** 时选择一个进化分支（不可逆）：

| 兵种 | 分支A | 分支B |
|------|-------|-------|
| 哥布林骑士 | **重甲猪骑**（防御+40%，嘲讽，牺牲输出） | **狂暴猪骑**（攻击+50%，攻击光环增强，更脆） |
| 哥布林炮手 | **火焰炮手**（AOE范围+60%，灼烧DOT） | **冰霜炮手**（命中减速30%，小概率冰冻） |
| 影魔 | **暗影领主**（全图控制时间+50%） | **噬魂者**（全图伤害+80%，控制减弱） |
| 皇家守卫 | **圣盾守卫**（格挡概率+30%，团队减伤） | **铁壁守卫**（自身硬度翻倍，牺牲团队增益） |

- 同一种兵种两种分支，玩家可能需要抽第二份来培养不同路线
- 增加收集深度和氪金点
- 新增配置：`evolution_branches.json`

#### 优化5：赛季制 + 通行证（Season Pass）
**为什么做**：赛季制是当前长线运营手游的标配（原神、崩铁、金铲铲），能有效解决"老玩家满级没事做"的问题，也是稳定收入来源。

**设计详情**：
- 每赛季持续 **6周**，赛季结束后部分重置（竞技场排名、赛季副本进度）
- **免费通行证**：40级奖励轨道，包含筹码、魂石、魔晶等常规资源
- **付费通行证**（30元/赛季）：额外40级奖励，包含限定皮肤、限定宝物、大量氪晶
- 赛季专属内容：
  - **赛季兵种**：每赛季新增1个限时兵种（如赛季1"龙族·火龙"，赛季2"机甲·铁卫"）
  - **赛季羁绊**：限时特殊羁绊效果（如"古代军团"羁绊，激活条件：上场至少3个不同赛季兵种）
  - **赛季排行榜**：推图层数/竞技场排名/远征记录，前100名获限定称号
- **赛季重置**：竞技场排名清零（保留历史最高），赛季副本关闭并开放新副本

#### 优化6：公会系统 + 领土战
**为什么做**：社交系统是提升留存的利器。原版缺乏社交元素，玩家容易"单机化"后流失。

**设计详情**：
- **公会基础**：创建/加入公会（上限30人），公会聊天，公会公告
- **公会Boss**（每日）：全公会协作攻击Boss，根据总伤害排名获得奖励
- **公会领土战**（每周六）：
  - 地图被划分为多个领土格子
  - 公会成员派兵攻打/防守领土
  - 占领领土产出持续资源（每小时氪晶、魔晶）
  - 一周结算，占领最多领土的公会用时获得赛季限定奖励
- **公会科技树**：公会成员共同捐献资源解锁全公会加成（如"全体攻击+5%"）
- **公会商店**：用公会币兑换稀有资源

#### 优化7：融合系统（Unit Merge）
**为什么做**：合成/融合是2024-2025年放置游戏热门机制（如Gold and Goblins），给多余的低品质兵种找到用途，减少"抽到重复"的挫败感。

**设计详情**：
- **同种融合**：3个同品质同兵种 → 1个高一品质的同一兵种
  - 例：3个绿色哥布林勇士 → 1个蓝色哥布林勇士
  - 有概率触发"变异"（+随机额外属性词条）
- **跨种融合**：特定组合产生新兵种
  - 例：哥布林骑士 + 哥布林炮手 + 特殊材料 → 融合兵种"哥布林战争机器"
- **融合配方**通过推图/副本/赛季逐步解锁
- 融合产生的兵种不可再分解
- 新增配置：`merge_recipes.json`

#### 优化8：无尽爬塔（Endless Tower）
**为什么做**：原版PVE内容有限（主线+5副本），消耗完就没目标。无尽模式提供永远有内容可刷的体验。

**设计详情**：
- **无尽之塔**：无限层数，每层敌人强度递增
- 每5层一个精英关，每10层一个Boss关
- 首次通关每层获得一次性奖励（氪晶、筹码）
- 每100层主题切换（哥布林关 → 魔族关 → 人族关 → 亡灵关）
- **全局排行榜**：全服最高层数排名，每周结算奖励
- **自动推塔**：设置阵容后可自动推到打不过的层（类似原版自动推图）
- 新增配置：`tower_floors.json`（模板配置，运行时动态生成具体数值）

#### 优化9：剧情系统 + 角色好感度
**为什么做**：原版几乎没有剧情和角色塑造，玩家对兵种没有情感连接。增加剧情和好感度能提升沉浸感和付费意愿。

**设计详情**：
- **主线剧情**：每章开头/结尾有剧情过场（文字+立绘对话），讲述魔王征讨四方的故事
- **兵种故事**：每个兵种有独立故事线（5章），通过使用该兵种战斗积累好感解锁
- **好感度系统**：
  - 好感度等级：陌生 → 熟悉 → 信任 → 契约 → 誓约
  - 每个等级解锁：故事章节 → 语音 → 皮肤 → 属性加成（小）
  - 送礼加速好感（消耗魔晶）
- **好感度加成**：誓约等级的兵种在战斗中有特殊出场动画和微弱属性加成（不影响平衡）

#### 优化10：实时PVP对战
**为什么做**：原版竞技场是打AI录像，缺乏真正的PVP体验。实时对战能极大提升竞技性和社交传播。

**设计详情**：
- **天梯模式**：实时匹配相近战力的玩家进行1v1对战
- 双方同时布阵（30秒倒计时）→ 揭示对方阵容 → 可调整一次（10秒）→ 战斗自动进行
- 战斗中可手动释放必杀技（配合优化3）
- **段位系统**：青铜 → 白银 → 黄金 → 铂金 → 钻石 → 大师 → 王者
- 每赛季根据段位结算奖励
- **观战系统**：可观看好友/排行榜玩家的对战直播
- **防作弊**：战斗计算在服务端进行，客户端只负责展示
